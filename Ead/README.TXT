Organização, Integração e Tutoriais (EAD) agrupados em site único em liguagem PHP

* Etapa 1
** Criação do layout base (Página Inicial)
** Formatação da estrutura para Layout Responsivo (Cores de Fundo - Temporárias para evidenciar a transição dos tamanhos de telas/browsers)

* Etapa 2
** Definição do Framework MVC (CodeIgniter)
** Configuração de acesso ao Banco de Dados
** Teste de rotinas básicas e acesso a base.

* Etapa 3
** Conclusão dos métodos de integração com a base de dados
** Utilizacao genérica da interface grafica, automatizada pelo banco de dados
** Inclusão de informações de EAD via banco de dados (Imagens, Textos e Videos)
